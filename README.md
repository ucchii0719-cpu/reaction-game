\# Reaction Time Game



\## Overview

This is a web-based reaction time game built using HTML, CSS, and JavaScript.



\## Features

\- Reaction time measurement

\- False start detection

\- Average score calculation

\- Best score tracking

\- Difficulty selection



\## Tech Stack

\- HTML

\- CSS

\- JavaScript

\- Gemini (AI)



\## Description

This project was developed using AI assistance and further improved with additional features and UI design.
